# 🏥 Smart Health Home Kit

A smart health monitoring system built with Flutter that connects to medical devices via Bluetooth (BLE) to track and analyze health data in real time.

---

## 🚀 Features

* 🔵 Bluetooth (BLE) device connection
* 📊 Real-time health data visualization (charts)
* 🧠 Smart health advice system (Rule-based)
* 🚨 Alerts for abnormal readings
* 📄 PDF report generation for doctors
* 👤 Multi-user profile support
* 🌍 Multi-language support (Arabic / English)
* 🔒 Local data storage with encryption (Hive)

---

## 🩺 Supported Devices

* Glucose Meter
* Blood Pressure Monitor
* Thermometer

---

## 📱 Screens

* Home Screen
* Device Details
* Charts & Analytics
* Alerts Screen
* Reports Screen
* Profile & Settings

---

## 🛠️ Tech Stack

* Flutter
* Bluetooth Low Energy (BLE)
* Hive (Local Storage)
* fl_chart (Charts)
* PDF generation

---

## 🧠 Smart System

The app includes a rule-based engine that provides:

* Instant advice after each measurement
* Daily health routines based on user profile
* Personalized alerts
